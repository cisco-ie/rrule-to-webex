> Converts iCalendar RFC-5545 recurrence rule to Cisco&#39;s WebEx recurrence XML tree


## Install

```
$ npm install --save ical-to-webex-recur
```


## Usage

```js
const icalToWebexRecur = require('ical-to-webex-recur');

icalToWebexRecur('unicorns');
//=> 'unicorns & rainbows'
```
